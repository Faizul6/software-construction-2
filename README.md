\# Software Construction 2



This repository contains my solutions for the \*\*Software Construction 2\*\* laboratory exercises at \*\*HAW Hamburg\*\*.



The labs focus on practicing fundamental Java programming concepts such as object-oriented programming, loops, conditionals, string processing, and method design.



\## Repository Structure



The repository is organized by labs and exercises:



\- `lab01/` – Introduction to classes, objects, and basic control structures

\- `lab02/` – Encapsulation, Methods, Arrays, and Static Variables

\- `lab03/` – Inheritance, Polymorphism, and Class Hierarchies

\- `lab04/` – Advanced Algorithms: Recursion, Backtracking, and GUI Programming

Each exercise contains its own implementation.
